# Online Course Platform - SQL Schema

## 📘 Description
This project builds the relational schema for an Online Course Platform using MySQL. It includes tables for users, courses, enrollments, categories, payments, and reviews.

## 📂 Files
- `create_schema.sql` – SQL script to create all tables and relationships
- `ER_Diagram.png` – Entity-Relationship diagram of the schema
- `README.md` – Project overview and instructions

## ⚙️ How to Use
1. Open MySQL Workbench
2. Connect to local server
3. Run `create_schema.sql`
4. Explore database and test relationships

## 🔧 Technologies
- MySQL 8.0
- MySQL Workbench
- GitHub
